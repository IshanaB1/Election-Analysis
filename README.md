# Election Intelligence System

## Overview

This project analyzes constituency-wise election data using Machine Learning.

The system consists of:

- Voter Turnout Forecasting
- Anti-Incumbency Prediction
- Swing Voter Analysis

## Dataset

Constituency-wise election result dataset containing:

- Candidate Information
- Votes Secured
- Elector Count
- Candidate Demographics
- Vote Percentages

## Models Used

### Model 1: Voter Turnout Forecasting

Algorithm:
- Linear Regression

Goal:
- Predict voter turnout percentage

### Model 2: Anti-Incumbency Prediction

Algorithm:
- Logistic Regression

Goal:
- Identify constituencies with high anti-incumbency risk

### Model 3: Swing Voter Analysis

Algorithm:
- Random Forest Classifier

Goal:
- Analyze factors influencing candidate victory
